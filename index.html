Como funciona a Web e o que é uma página web?

Você deve ter notado o uso do termo "conteúdo web" ao invés de "página web"; isso é intencional. Apesar de falarmos de "visitar uma página web", o que realmente queremos dizer é algo como "olhar para todo o conteúdo de texto e imagens em um endereço no nosso computador". O texto que nós lemos e as imagens que vimos são renderizados pelo navegador web, que recebem certas instruções encontradas em arquivos individuais.

Quais são os papéis das linguagens HTML e CSS numa página web?

Esses arquivos contêm texto marcado ou cercado por códigos HTML que informam ao navegador a estrutura do texto - como título, parágrafo, citação e assim por diante. Algumas marcações HTML dizem ao navegador para exibir um arquivo de imagem ou vídeo em vez de texto simples, o que nos traz de volta a este ponto: Diferentes tipos de conteúdo são enviados ao seu navegador da Web, então simplesmente dizer que a página da Web não começa a cobri-lo. Aqui, usamos o termo conteúdo da web para abranger toda a gama de texto, imagem, áudio, vídeo e outras mídias encontradas online.

Outros arquivos contêm CSS que informa ao navegador como uma página deve ser - texto em negrito, com borda azul, cor vermelha e assim por diante. CSS define como o navegador deve exibir a estrutura de conteúdo definida no HTML.

O que são ambientes de programação e com HTML/CSS?

Ambientes de programação, como o próprio Repl.it por exemplo, são plataformas instaláveis ou online compostas de vários programas utilizados para emular em ambiente seguro para teste de diversas linguagens de programações diferentes. Diferentemente de linguagens de programação mais robustas, como C#, Python e outras, HTML e CSS são linguagens de marcação, ou seja, elas dão estrutura e estilo a um texto que será apresentado em uma tela.

Pela sua simplicidade, linguagens de marcação não requerem necessariamente um ambiente de programação para serem desenvolvidas e testadas.
